# Trackerify
Kanban board app with Vuejs client and Nodejs/Express server
![Preview](https://imgur.com/rdpd2x2 "Board Page")

## Features
* Authentication
* Drag and Drop
* Real time collaboration
* Responsive design

### Made with these technologies
#### Server side
* Nodejs
* Express
* MongoDB
* Mongoose
* Socket.io
* JWT
#### Client side
* Vuejs
* Vuex
* Vuex-persistedstate
* Vue router
* Vuedraggable
* Vue-Socket.io
* Vue-color
* Portal-vue
* Axios
* Tailwindcss
* Moment
* JWT-decode
